PM機器保養基本構造&原理

專用術語

MFC=天然氣流量控制器
PCW (S)Supply(冷進)
PCW (R)Return(冷出)


機台維護定期保養操作流程

檢查(壓力/流量)⮕手動模式⮕System setting⮕Heat jacket⮕關加熱帶電源待溫度低於60℃⮕冷卻後關閉水路、氣體閥門
M6六角拆下護蓋⮕拆除馬達刮刀電源線⮕拆除Innet管道加熱帶電源線⮕拆氣管及噴嘴檢查清理⮕拆卸刮刀
拆除燃燒器Sensor*2⮕CTW冷水管拔除⮕螺絲*4擰鬆⮕拔除燃燒器⮕拆除C型Clamp⮕拆卸燃燒器


燃燒的定義：

可燃性燃料, 氧氣, 點火源(發火點以上的溫度)
150~200℃用來燃燒分解氣體，經高溫分解冷卻後排出。


水質種類

LMP(流量單位)
CW(city water)自來水
PCW(cooling water)冷水
DW(Drain water) 廢水
PCW S (冷入)
PCW R (冷出)


循環水供應三部位：

(1)Swirl (2)Quench (3)Wet


Burner－上蓋內部構造(燃燒器)

Burner Plate (供LNG，O2，CDA氣體流入形成火焰)
Inlet Gas Port 毒氣進氣口
Teflon Guide 鐵氟龍套管(防止腐蝕、堵塞)
Flame Sensor 火焰感應器*2
GN2氮氣(防止Sensor過熱)
點火裝置
LNG天然氣(助燃)
O2氧氣(助燃)
CDA空氣(助燃)
PCW (S)(冷入)
PCW (R)(冷出)
Burner Temp swich 燃燒器溫度感應器
(燃燒器溫度達150℃以上接點分離)


Reactor－反應爐內部構造

Y - Reactor

危害氣體在Y-Reactor上部燃燒，經過Quench冷卻，再通過螺旋構造進行Wet冷卻

Swirl Guide(保護環防止水滴飛濺)
Swirl Water Line(持續供水冷卻&防粉塵堆積，兩處噴嘴)
Atomizing Nozzle(二流體加速水處理效率，噴水速度慢)
Wet Nozzle(中間循環水冷卻氣體，兩處噴嘴)
Quench Nozzle(下方循環水冷卻氣體，三處噴嘴)
Reactor Temp 反應爐溫度感應器
(Quench冷卻失效則反應爐達70℃以上時接點分離)


Wet Tower－加濕管內部構造

(1)Exhaust Elbow(排氣彎頭) CW噴水凝結完粉塵通過排水口流入 Tank
(2)Exhaust Reducer(排氣減速) 提高水處理效果的Packing zone
(3)Exhaust Top Bellows(頂部排器波紋管) CW降溫處理完氣體排出處
3-1 Exhaust Pressure Sensor Port（排气管道壓力Sensor&氮氣降溫Sensor）
3-2 Exhaust Temp Sensor（排氣管溫度Sensor）
3-3 CW Supply Nozzle（CW噴嘴）降溫作用


Wet Tank－廢水內部構造

Tank Mesh Filter 過濾網(防止噴嘴、熱水交換器堵塞)
Circulation Pump 循環泵(供循環水進入設備)
HCS [Hydro Cyclone System] 離心分離機(異物粉塵分離)
Drain Line 排水管(空壓閥門感應後，Tnak內部廢水排出)
Heat Exchange 熱水交換器(PCW與循環水交叉冷卻裝置)
Manifold 流量計(熱水交換後，進設備前流量檢測)
C/W Temp Sensor 溫度計(熱水交換後，循環水溫度)
Swirl Flow Sensor 流量感應器(低於6LPM設備警報)
Wet Nozzle Flow Sensor 流量感應器(低於6LPM設備警報)
Quenching Flow Sensor 流量感應器(低於6LPM設備警報)
BCW Flow Sensor 常用水流量感應器(Ventur ：1 LPM以下 Exhaust:1.5 LPM以下設備警報)
S1S2S3 Level Sensor 水位偵測
S5 Level Sensor 最高水位偵測
Tank Temp Sensor Tank內部溫度計(超過50℃警報)


右側構造

LNG Line 天然氣供應線

(1)LNG Manual Valve 手動閥門
(2)LNG Solenoid Valve LNG電磁閥
(3)LNG MFC[Mass Flow Controller] 天然氣流量控制器
(4)LNG Solenoid Valve 副LNG電磁閥

Main CDA Line 空氣供應線

(1)CDA Manual Valve 手動閥門
(2)CDA Regulator 壓力調節器(CDA壓力表，可手動調節)
(3)CDA MFC[Mass Flow Controller] 空氣流量控制器
(4)CDA Solenoid Valve CDA電磁閥

O2 Line 養氣供應線

(1)O2 Manual Valve 手動閥門
(2)O2 Regulator 壓力調節器(O2壓力表，可手動調節)
(3)O2 MFC [Mass Flow Controller] 氧氣流量控制器
(4)O2 Pneumatic Valve 氧氣氣動閥門

Main N2 Line 氮氣供應線

(1)N2 Manual Valve 手動閥門
(2)N2 Regulator 壓力調節器(N2壓力表，可手動調節)
(3)N2 Solenoid Valve N2電磁閥


背部構造

PCW-Supply Line PCW供應線

(1)PCW-S Manual Valve 手動閥門
(2)PCW-S Pneumatic Valve 氣動閥門

PCW-Return Line PCW排出管

(1)PCW-R Manual Valve 手動閥門
(2)PCW Leak Check Gauge 漏水檢測表
(3)PCW Flow Sensor 流量感應器(低於25LPM設備警報)

CW Line 常用水供應線

(1)BCW Manual Valve CW手動閥門
(2)BCW Regulator 壓力調節器(BCW壓力表，可手動調節)
(3)BCW Leak Check Gauge 漏水檢測表(依壓力變化確認是否漏水)
(4)BCW Pneumatic Valve BCW氣動閥

Drain Line 廢水排管

Drain Manual Valve 手動閥門

背部底下電箱

Pneumatic Valve [SOL 1] 氣動閥控制開關
Solenoid Valve [SOL 2] 電磁閥控制開關
Mini Heat Exchange [BCW] 熱水交換器

2.反應爐溫度感應器Reactor Temp，Quench冷卻失效則反應爐超過70℃時接點分離
3.排氣管溫度感應器Exhauht Temp，超過40℃警報
